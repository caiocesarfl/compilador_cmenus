Para executar a aplicação basta descompactar o arquivo “analisadorLexico.zip”, abrir o terminal do seu sistema operacional, navegar até a pasta onde o arquivo foi descompactado e digitar:

java – jar analisadorLexico.jar [caminho arquivo de código] 

Exemplo 1 (para terminal Linux):
java – jar analisadorLexico.jar testes/teste_fornecido.cmm

Exemplo 2 (para terminal Windows):
java – jar analisadorLexico.jar testes\teste_fornecido.cmm
	 Foi atribuído a extenção .cmm para os arquivos, porém é possível qualquer qualquer arquivo de texto fornecido.
